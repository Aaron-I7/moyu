# 国际化产品重构说明

## 1. 重构目标

本次重构将原先面向中国打工年轻人的碎片消遣应用，升级为面向海外年轻职场人群的国际化产品，核心目标如下：

- 完整多语言体系与动态切换能力
- 冷色系极简视觉系统与无障碍增强
- 面向 GDPR/CCPA 的隐私同意机制
- 符合国际使用场景的内容推荐与防沉迷提醒
- 支持 RTL（阿拉伯语）与双向文本渲染

## 2. 已完成改造清单

### 2.1 国际化能力

- 建立 `src/core/i18n` 国际化核心模块，支持 `en/es/ar/zh` 语言包动态加载。
- 新增语言元数据（地区、时区、货币、度量制、RTL 标记），切换语言时自动设置 `document.lang` 与 `document.dir`。
- 路由标题接入国际化 key，页面标题随语言切换同步更新。
- 模块配置新增 i18n 字段，支持模块名称、描述、标签按语言显示。

### 2.2 视觉与交互重构

- 主题系统升级为冷色系设计（Arctic/Slate/Midnight），满足欧美简洁风格。
- 全局字体升级为 Inter + Roboto，支持动态字号缩放。
- 增强键盘焦点样式和对比度，提升 WCAG 2.1 可访问性基础体验。
- 增加手势导航（移动端左右滑动切换主栏目）。
- 增加语音导航入口（浏览器支持时可通过语音快速跳转频道）。
- 左侧工作台导航保持左舵习惯，移动端改为左下角可达位置。

### 2.3 内容与合规改造

- 新增本地化内容策略引擎（节假日、通勤时段、地区工作节奏）并在首页展示动态建议。
- 新增 GDPR/CCPA 风格隐私授权横幅，可选择分析、个性化、社交分享权限。
- 新增社交分享入口，接入 X/TikTok/Instagram 外链分享流程与系统分享降级。
- 新增连续使用时长提醒（欧盟区域 45 分钟触发，其他区域 60 分钟触发）。

### 2.4 技术标准对齐

- 统一 UTF-8 文本处理链路，支持中英文、西语、阿语混排。
- `ar` 语言自动启用 RTL，支持 UI 布局镜像关键路径。
- 新增国际化测试矩阵文件：`src/core/i18n/testMatrix.ts`，覆盖语言、时区、货币、度量单位、RTL。
- WebSocket 地址从硬编码切换为环境变量优先与安全默认策略。

## 3. 性能目标与当前策略

### 3.1 冷启动 < 2s

- 仅默认加载 `en` 语言包，其它语言包按需动态加载。
- 保持应用壳层轻量，减少首次阻塞资源。

### 3.2 语言包动态加载 < 500ms

- 使用 Vite 动态分包 `import()` 加载语言资源。
- 语言包内容按页面壳层优先，模块内深层文案后续分批继续迁移。

## 4. 后续建议（下一阶段）

- 将子模块内部文案（游戏/工具详细界面）全部迁移到 i18n 资源。
- 为分享能力补齐 TikTok/Instagram SDK 级对接（当前为 Web 入口级对接）。
- 建立 Playwright 自动化本地化回归：语言切换、RTL 布局、隐私授权流程、长时使用提醒。
- 引入真实地区节假日 API，替代当前规则引擎中的静态节日映射。
