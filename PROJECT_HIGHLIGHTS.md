# 摸鱼平台 - 项目亮点文档

## 项目概述

摸鱼（Moyu）是一个基于 Vue 3 + TypeScript + Vite 构建的现代化休闲办公平台，集成了多种放松、专注和实用工具功能。项目采用模块化架构设计，支持国际化（中/英），提供丰富的交互体验。

---

## 核心亮点功能

### 1. 像素钓场 - 沉浸式像素钓鱼游戏

![像素钓场](./screenshots/pixel-fishing.png)

**一句话描述**：基于 PixiJS 的沉浸式像素风钓鱼游戏，5大钓点与40种鱼类可探索。

**亮点详情**：
- 采用 PixiJS 8.x 渲染引擎，实现流畅的像素风格动画
- 5个特色钓点：溪流、河流、湖泊、海岸、深海
- 40+ 种可收集鱼类，支持鱼类图鉴系统
- 创新的时机小游戏（Timing Bar）钓鱼机制
- 完整的音效系统：场景BGM、环境音、钓鱼音效
- 支持键盘快捷键操作，提升游戏体验

---

### 2. 春风解签 - AI 驱动的周易占卜

![春风解签](./screenshots/divination.png)

**一句话描述**：结合 AI 与传统周易的六爻占卜系统，提供诗意的解卦指引。

**亮点详情**：
- 真实的金钱卦六爻起卦系统
- 集成 Supabase Edge Functions 调用 AI 进行解卦
- 《剑来》风格的诗意解卦文案
- 精美的风铃动画和铜钱投掷效果
- 支持一键起卦和逐爻投掷两种模式
- 完整的卦象展示和变爻标注

---

### 3. 番茄专注钟 - 沉浸式专注计时器

![番茄专注钟](./screenshots/pomodoro.png)

**一句话描述**：结合白噪音的沉浸式番茄工作法计时器，提升专注效率。

**亮点详情**：
- 完整的番茄工作法实现：专注、短休息、长休息循环
- 集成白噪音混音系统，支持多音轨叠加
- 任务输入与追踪功能
- 可自定义专注时长和休息时长
- 优雅的圆形进度条动画
- 支持全局播放，切换页面时继续计时

---

### 4. 电子宠物 - 工作日的治愈陪伴

![电子宠物](./screenshots/virtual-pet.png)

**一句话描述**：在工作日照顾一位小小陪伴者，增添工作乐趣。

**亮点详情**：
- 三种可爱宠物可选：猫咪、狗狗、兔子
- 完整的养成系统：饥饿、快乐、精力、清洁度
- 工作时间检测，自动切换摸鱼/休息状态
- 丰富的互动动作：喂食、玩耍、抚摸、睡觉、洗澡
- 互动日志记录，追踪宠物成长
- 本地持久化存储，数据不丢失

---

### 5. 敲木鱼 - 节奏解压神器

![敲木鱼](./screenshots/wooden-fish.png)

**一句话描述**：通过节奏敲击调节呼吸，快速缓解压力。

**亮点详情**：
- 真实的木鱼敲击音效
- 功德值累计系统，浮动文字动画
- 支持自动敲击模式，可调节速度
- 可选震动反馈（移动端）
- 支持键盘空格键操作
- 简洁优雅的视觉设计

---

### 6. 万年历 - 智能日历与备忘

![万年历](./screenshots/calendar.png)

**一句话描述**：集成农历、节气、黄历的智能日历，支持多地区节假日。

**亮点详情**：
- 完整的农历转换与显示
- 24节气自动标注
- 多地区节假日支持：中国、新加坡、马来西亚、韩国、日本
- 黄历信息：宜忌、冲煞、五行、干支
- 云端备忘录功能（需登录）
- 双击/右键快速添加备忘
- 距离周末倒计时显示

## 技术架构亮点

### 模块化设计
- 统一的模块注册系统（ModuleRegistry）
- 支持动态加载和卸载模块
- 模块配置化，易于扩展

### 国际化支持
- 完整的中英文双语支持
- 使用 Vue I18n 实现
- 所有用户可见文本均已国际化

### 主题系统
- 支持多主题切换：日间、夜间、像素风
- CSS 变量驱动的主题切换
- 平滑的主题过渡动画

### 响应式设计
- 完美适配桌面端和移动端
- 流式布局与自适应网格
- 触摸友好的交互设计

## 总结

摸鱼平台是一个功能丰富、设计精美的休闲办公平台，通过模块化的架构设计，将多种放松和实用工具整合在一起。项目采用现代化的前端技术栈，注重用户体验和交互设计，是 Vue 3 生态下的优秀实践案例。
