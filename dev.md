# 摸鱼吧开发指南

本 Skill 为摸鱼吧（微放松平台）项目提供开发指导和规范。

## 项目定位

摸鱼吧是一个轻量化微放松平台，面向 18-35 岁职场人和学生，满足碎片化放松需求（5-15 分钟使用场景）。

**核心原则**：轻量化、低门槛、高复用、有新鲜感

## 技术栈

| 类别 | 技术选型 |
|-----|---------|
| 框架 | Vue 3 + TypeScript |
| 构建 | Vite 5 |
| UI 库 | Naive UI |
| 状态管理 | Pinia |
| 路由 | Vue Router 4 |
| 游戏引擎 | PixiJS 8 |
| 动画库 | GSAP |
| 音频库 | Howler.js |
| 日期处理 | dayjs |
| 本地存储 | IndexedDB + localStorage |

## 核心模块开发规范

### 游戏模块规范

1. **性能要求**
   - 首屏加载 < 3 秒
   - 动画帧率 ≥ 60fps
   - 资源按需加载

2. **交互规范**
   - 无时间限制（除特殊玩法）
   - 答错/失败无惩罚
   - 连续成功有小奖励反馈

3. **状态持久化**
   - 游戏进度保存到 IndexedDB
   - 支持断点续玩

### 工具模块规范

1. **白噪音工具**
   - 支持多音源混合播放
   - 音量独立控制
   - 定时关闭功能（5-60 分钟）

2. **计时器工具**
   - 简洁界面，无多余功能
   - 柔和提醒音
   - 支持自定义时长

3. **壁纸生成工具**
   - 使用 html2canvas 导出
   - 支持自定义文字和图案

### 阅读模块规范

1. **小说渲染**
   - 支持自定义字体、背景、字号
   - 阅读进度本地存储
   - 弹窗隐藏式阅读模式

2. **文件处理**
   - 前端直接解析 txt 文件
   - FileReader API 读取本地文件

## 开发优先级

**第一阶段（核心）**：
1. 白噪音工具
2. 消消乐游戏
3. 解压球/捏捏乐
4. 计时器工具

**第二阶段（益智）**：
1. 拼图游戏
2. 成语接龙
3. 数字填色

**第三阶段（养成）**：
1. 虚拟盆栽
2. 极简养宠

**第四阶段（阅读+工作）**：
1. 小说阅读器
2. 工作类工具（JSON 格式化、时间戳转换、GUID 生成）

## 代码规范

1. 使用 Composition API 编写组件
2. TypeScript 严格模式
3. 统一使用 Pinia 管理状态
4. 遵循 Vue 3 风格指南

## 性能优化策略

1. 路由懒加载
2. 关键资源预加载
3. 静态资源 CDN 托管
4. Gzip/Brotli 压缩
5. 虚拟列表处理长列表
6. requestAnimationFrame 处理动画
